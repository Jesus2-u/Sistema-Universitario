# Sistema-Universitario

Este proyecto consiste en el desarrollo de un sistema de control de asistencia para estudiantes universitarios. El objetivo principal es registrar automáticamente el ingreso de los alumnos a la universidad, almacenando la información de asistencia de manera diaria y organizada.

El sistema permitirá generar registros de entrada por cada estudiante, manteniendo un historial de asistencias que podrá ser consultado posteriormente para fines administrativos y académicos.

# Objetivos

- Registrar el ingreso de estudiantes a la universidad.
- Almacenar los registros de asistencia diariamente.
- Mantener un historial de asistencias por alumno.
- Facilitar la consulta y control de asistencia estudiantil.

# Funcionalidades Principales

- Registro de estudiantes.
- Registro de ingreso y asistencia.
- Almacenamiento de fecha y hora de entrada.
- Consulta de historial de asistencias.
- Gestión básica de información de alumnos.

# Tecnologías a Utilizar

- GitHub para gestionar el desarrollo del proyecto.
- Java como Lenguaje de programación.
- SQL Server para la Base de datos .
- Justmind para el prototipado para las interfaces.
